# presupuesto

Aplicación web para gestionar ingresos y egresos, visualizar balance
disponible y analizar datos mediante gráficos.

## Tecnologías
- HTML5
- CSS3
- JavaScript (ES Modules)
- LocalStorage
- Chart.js

## Funcionalidades
- Agregar y eliminar transacciones
- Balance automático
- Persistencia de datos
- Gráfica de barras para visualizar ingreso y egresos
- Modo oscuro

## Arquitectura
- Estado centralizado
- Separación de responsabilidades
- UI desacoplada de lógica
- Persistencia encapsulada

## Aprendizajes
- Uso de ES Modules
- Delegación de eventos
- Refactor a estado único
- Arquitectura escalable
- Integración de librerías externas (Chart.js)
- Control del layout y comportamiento responsive de gráficos
